String & Array Methods
Method Description Example
toCapital() String'ni 1-harfini bosh harfga o'tkazib beruvchi metod 'capital'.toCapital() --> Capital
toWaveCase() String'ni juft harflarini katta, toq harflarini kichik harflarga o'tkazib beruvchi metod 'HELLO world'.toWaveCase() --> hElLo wOrLd
sumAll() Arrayni elementlarini bir-biriga +, -, *, / beruvchi metod sumAll([1,2,3] + [4,5,6]) --> 21
multiplyTo() Arrayni elementlarini berilgan son darajasiga oshirib beruvchi metod [1,2,3,4].multiplyTo(3)
powerTo() Arrayni elementlarini berilgan son darajasiga oshirib beruvchi metod [1,2,3,4].powerTo(3)
